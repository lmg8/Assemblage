Getting Started

Download the library from Github.

To use Assemblage Library, add the following stylesheet and scripts to your webpage: 

    <!--Library Style Sheet-->
    <link rel="stylesheet" type="text/css" href="Assemblage_styles.css">
    <!--Load jquery from Google CDN.-->
    <script defer src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
    <!--Load Assemblage Library-->
    <script defer type="text/javascript" src='./js/Assemblage.js'></script>

Basic Functionality

This example shows how to initialize the library and the bare minimum to create an Assemblage drawer.
It also shows how to assign an onClick event on a button you created, which will open the Assemblage drawer. 


    const a = assemble();
    a.createAssemblage({});
    //buttonClass is the class name of the button you created to open Assemblage
    a.openDrawer("buttonClass");

The following is an example of a button to open Assemblage: 


    <span class="buttonClass">&#9776; Click me to try it out</span>
☰ Click me to try it out
*Note: If you would like the Assemblage drawer to not overlay your webpage, but instead to push your page content like the main page example,
wrap your webpage body content in a div with id name "main" like so: 
 <body>
    <div id="main">
        <!--You're main content goes here-->
    </div>
    <div>
        <!--Assemblage drawer will go here-->
    </div>
 </body>

Currently, Assemblage saves to the clientside's localStorage.
The following code will show how to save the client's folders before they navigate away from your webpage. 


    const a = assemble();
    if (a.getFolders() === null) {
        a.createAssemblage({})

    } else {
        a.reAssemblage({}, e.getFolders())
    }
    a.openDrawer("buttonClass");

Functions returned from assemble() 

createAssemblage(settings)

This function creates an empty Assemblage drawer with the following settings object example:

 settings = {title: "Title to give Assemblage Drawer", width: "350px", position: "right"}

This function accepts an empty object {} or any one of the above settings.
title is of type string. Default value: "Assemblage"
width is of type string. Default value: "350px".
  Accepts any CSS width value. The recommended minimum width is 300px.
position is of type string. Default value: "left". Other value accepted is "right".
  This positions the drawer either on the left or right side of the page. 

openDrawer(buttonClass)

This function adds an onClick event handler to the buttonClass.
buttonClass is of type string. It's the class name of the button to add the event handler to. 

reAssemblage(settings, folders)

This function recreates an Assemblage drawer with the following settings and list of folders:

 settings = {title: "Title to give Assemblage Drawer", width: "350px", position: "right"}
 folders = [folderObject]

This function accepts an empty settings object {} or any one of the above settings.
title is of type string. Default value: "Assemblage"
width is of type string. Default value: "350px".
  Accepts any CSS width value. The recommended minimum width is 300px.
position is of type string. Default value: "left". Other value accepted is "right".
  This positions the drawer either on the left or right side of the page.
folders is a list of folder objects. 

getFolders()

Returns: a list of folder objects retrieved from the clientside's localStorage. 

clearFolders()

This function removes the list of folder objects retrieved from the clientside's localStorage. 

deleteFromDOM()

This function removes the Assemblage drawer from the DOM.

changeHeaderBackgroundColor(color)

This function changes the header background color of the Assemblage drawer.
color is of type string. Accepts a valid css color. 

changeTitleFontColor(color)

This function changes the header title font colors of the Assemblage drawer.
color is of type string. Accepts a valid css color. 

changeSecondaryFontColor(color, hoverColor)

This function changes the font color of the "create new folder" and "save current page" button Assemblage drawer.
It also changes the color when you hover over the button.
color is of type string. Accepts a valid css color.
hoverColor is of type string. Accepts a valid css color.

changeHeaderButtonColor(color, hoverColor)

This function changes the font color of the navigation (exit and back) buttons in the header of the Assemblage drawer.
It also changes the color when you hover over the button.
color is of type string. Accepts a valid css color.
hoverColor is of type string. Accepts a valid css color.

changeBackgroundColor(color)

This function changes the background color of the Assemblage drawer.
color is of type string. Accepts a valid css color. 

changeFolderBackgroundColor(color)

This function changes the folders' background colors of the Assemblage drawer.
color is of type string. Accepts a valid css color. 

changeFolderButtonColor(color, hoverColor)

This function changes the font color of the delete and edit buttons in the folders of the Assemblage drawer.
It also changes the color when you hover over the button.
color is of type string. Accepts a valid css color.
hoverColor is of type string. Accepts a valid css color.

changeFolderFontColor(color)

This function changes the folders' font colors of the Assemblage drawer.
color is of type string. Accepts a valid css color.
